This is the "Basic Requirements" page of WebTrilinos

Please check the following:

  1. The first thing you need is an installed (and working!) version of Trilinos. The following packages must be installed: Teuchos, Epetra, EpetraExt, Amesos, IFPACK, AztecOO, ML. You also need to enable PyTrilinos by adding --enable-python to your Trilinos configure script. Finally, you must have installed Trilinos by typing make install at the top Trilinos directory.

  2. You need PHP installed on your system. PHP must process both .php and .html files. The line below should not contain any <? or ?>. If it does, check your web server installation, to be sure that PHP takes care of .html as well.



  3. You need convert installed on your system. Below you have the location of convert as defined by /bin/env.

  4. You need PyChart to plot results in the MatrixPortal module. If you don't have it, you can still use MatrixPortal and the other modules, but without graphs. PyChart can be downloaded at the web page http://home.gna.org/pychart/. Below you have the output of python -c "from pychart import *". If nothing is reported, PyChart is correctly installed on your system.
